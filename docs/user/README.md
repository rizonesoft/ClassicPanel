## Features

- **Classic Interface**: Traditional Control Panel interface with modern enhancements
- **Direct Access**: Direct access to 100+ system settings and utilities
- **Modern Enhancements**: Real-time search, favorites, command palette, themes
- **Fast Performance**: Sub-second startup, optimized for speed
- **Multiple View Modes**: Large Icons, Small Icons, List, and Details views
- **Comprehensive**: All Windows Control Panel items
- **Portable**: Single-file executable, no installation required

## System Requirements

- Windows 10 or Windows 11
- No .NET runtime installation required (uses Native AOT)
