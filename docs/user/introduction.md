# Introduction to ClassicPanel

## What is ClassicPanel?

**The power of direct control, refined.**

ClassicPanel is a comprehensive, fast, intuitive, and beautiful Control Panel replacement for Windows 10 and Windows 11. ClassicPanel brings back the familiar Windows Control Panel interface with modern enhancements, providing direct access to 100+ system settings and utilities, refined for speed, search, and seamless navigation.

## Why ClassicPanel?

With the introduction of Windows 8 and later, Microsoft replaced the traditional Control Panel with the modern Settings app. While the Settings app offers a modern interface, many users prefer the classic Control Panel's simplicity and direct access to system settings.

ClassicPanel brings back that classic interface with significant improvements, allowing you to:
- **Direct Access**: Access 100+ Control Panel applets and system settings in a familiar interface
- **Modern Enhancements**: Real-time search, favorites system, command palette, and beautiful themes
- **Fast Performance**: Sub-second startup, optimized for speed and responsiveness
- **Portable**: Single executable file, no installation required
- **Comprehensive**: All Windows Control Panel items including Hardware, Security, System, Network, Programs, Appearance, and more

## Key Features

- **Classic Interface**: Traditional ListView-based interface that feels familiar
- **Direct Control**: Direct access to 100+ system settings and utilities
- **Modern Enhancements**:
  - Real-time search with fuzzy matching
  - Favorites and Quick Access system
  - Command Palette (Ctrl+K) for quick actions
  - Category-based sidebar for intuitive navigation
  - Light/Dark/High Contrast themes with smooth animations
- **Multiple View Modes**: Large Icons, Small Icons, List, and Details views
- **Comprehensive**: All Windows Control Panel items (Hardware, Security, System, Network, Programs, Appearance, Mobile, Backup, and more)
- **Fast Performance**: Sub-second startup, optimized for speed and responsiveness
- **Portable**: Single executable file, no installation needed
- **Standalone**: .NET runtime is bundled in the executable with ReadyToRun for faster startup - no separate .NET installation needed
- **Extensible**: CPL extension system for developers to add custom applets

## System Requirements

- **Operating System**: Windows 10 or Windows 11
- **Architecture**: x64 (64-bit)
- **Additional Software**: None required - the .NET runtime is included in the executable, so you don't need to install .NET separately

## Getting Started

1. Download ClassicPanel
2. Extract or run the executable
3. Place .cpl files in the `system` folder (created automatically)
4. Launch ClassicPanel and browse your applets

See the [Installation Guide](installation.md) for detailed instructions.
